CSS 이해하기
HTML이 구조를 만드는 것이라면 CSS는 그 내부를 예쁘게 디자인하는 역할을 해요. 아래와 같이 선택자를 활용해 특정 태그를 선택해서 디자인을 적용할 수 있답니다.
id 선택자
id는 문서 내에서 고유해야 합니다. 다시 말하면, 한 HTML 문서에서 같은 id를 가진 태그는 하나만 있어야 합니다! 특정 요소를 유일하게 식별할 때 유용하겠죠?
CSS
복사
/* id 선택자를 사용하는 경우, #을 사용한다는 점! */ #idName { color: white; /* HTML 문서에서 idName이라는 아이디를 가진 태그에 흰색 폰트 컬러를 적용해줘! */ }
​
class 선택자
고유한 하나의 태그가 아니라 동시에 여러개의 태그에 디자인을 적용해야 할 수 있잖아요? 이때는 class선택자를 사용할 수 있어요. 여러 태그에 공통적인 스타일을 적용하고 싶을 때 사용하면 편리합니다.
CSS
복사
/* class 선택자를 사용하는 경우, .을 사용한다는 점! */ .className { width: 100px; height: 50px; background-image: url('./image.png'); background-size: cover; }
​
태그 선택자
앞서 배웠던 <div>, <p>, <h1> 등의 태그에 공통적인 스타일을 일괄 적용하고 싶을 수도 있겠죠? 이때는 태그를 선택해서 작성해주면 됩니다.
CSS
복사
p { font-size: 10px; /* 모든 p 태그에 10px의 폰트 사이즈와 흰색 폰트 컬러를 적용해줘! */ color: white; }
​
